# עדכון FitKeeper — גל 1 + גל 2

מדריך הטמעה של הפיצ'רים החדשים. כל הקבצים המעודכנים בחבילה הזו.

---

## מה חדש

### גל 1 — שיפור חוויית אימון
- **תבניות אימון** (Routines) — צור שגרה פעם אחת, הפעל אותה בלחיצה
- **Auto-fill חכם** — פתיחת תבנית מביאה אוטומטית את המשקלים והחזרות מהפעם האחרונה
- **גרף התקדמות** לכל תרגיל — בחר תרגיל, ראה משקל/חזרות לאורך זמן
- **תובנות שבועיות** — אימונים השבוע מול שבוע שעבר, ממוצע, שיאים אישיים

### גל 2 — אוכל מהיר ואירובי
- **ארוחות שמורות** — בנה "הבוקר שלי" פעם אחת, הוסף ליום בקליק
- **אימוני אירובי** — עמוד נפרד עם שדות ייעודיים (משך, מרחק, קצב)
- **חישוב קלוריות אוטומטי** — לפי נוסחת MET ומשקל שלך
- **גרף התקדמות אירובי** — קצב ריצה/משך לפי פעילות

## הטמעה בשני שלבים

### שלב 1 — הרץ SQL ב-Supabase

1. היכנס ל-Supabase → SQL Editor → New query
2. פתח את הקובץ `supabase/upgrade.sql` מהתיקייה
3. העתק הכל → הדבק → Run

אתה אמור לראות `Success. No rows returned`. זה מוסיף 5 טבלאות חדשות בלי לפגוע בקיימות.

### שלב 2 — העלה קבצים ל-GitHub

יש שתי אפשרויות: דרך הדפדפן או דרך Codespace.

**דרך Codespace (מומלץ, קל יותר):**
1. פתח את ה-Codespace שלך
2. גרור את כל תוכן התיקייה החדשה אל ה-Explorer משמאל
3. כש-VS Code שואל "Replace existing files?" — בחר **Yes to All**
4. בטרמינל הרץ:
   ```
   git add .
   git commit -m "Gal 1 + Gal 2 features"
   git push
   ```

**דרך GitHub ישירות:**
לכל קובץ שהשתנה/נוצר, פתח אותו ב-GitHub, לחץ על העיפרון, הדבק את התוכן החדש, Commit.
(לא מומלץ — יש עשרות קבצים)

## אחרי העלאה

Vercel יזהה אוטומטית ויבנה מחדש. תוך ~2 דקות האפליקציה עולה עם כל הפיצ'רים החדשים.

## בעיות נפוצות

**"relation does not exist"** בדשבורד אחרי עדכון
→ לא הרצת את ה-SQL. חזור לשלב 1.

**"Module not found"** בבנייה של Vercel
→ קובץ לא הועלה. בדוק שכל הקבצים ברשימה מעלה קיימים ב-GitHub.

**התבניות/ארוחות לא מופיעות אחרי יצירה**
→ רענן את העמוד. נכון שבעתיד אפשר לבנות refresh אוטומטי.

## איך להשתמש

### תבניות אימון
1. לך לעמוד **אימון** → לחץ **📋 תבניות**
2. צור תבנית, תן שם, בחר קבוצת שרירים
3. הוסף תרגילים עם מספר סטים וחזרות יעד
4. כשתתחיל אימון, בחר "מתבנית" → בחר תבנית → הכל מוכן

### ארוחות שמורות
1. בעמוד **תזונה** → לחץ **🍽️ ארוחות שמורות**
2. צור ארוחה ("הבוקר שלי"), הוסף מזונות
3. בכל יום, בעמוד תזונה, לחץ **⚡ הוסף ארוחה ליום** → בחר ארוחה → הוספה בקליק

### אירובי
1. בעמוד **אימון** → לחץ **🏃 אירובי**
2. "רשום אימון אירובי" → בחר פעילות, משך, מרחק אופציונלי
3. הקלוריות מחושבות אוטומטית לפי המשקל שלך
4. גלול למטה לראות גרף התקדמות לפי פעילות
